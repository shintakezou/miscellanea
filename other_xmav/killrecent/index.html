Kill recents (KDE)

It seems like KDE spread recents file almost everywhere... how to kill them all? This script's purpose is this, but of course it can fail. Just try it.

I am not responsible for any data loss due to right or wrong use or peruse or misuse of this script! Watch your step!

How does it work? It simply looks inside the kind of file KDE can store recent information in, and then strip the line(s) containing the recent information.

It does a backup you can of course remove. It looks inside $HOME/.kde/share/config for files like XXXrc and XXX.new ... then it checks inside kdeglobals too.
